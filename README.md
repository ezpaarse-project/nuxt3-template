# Nuxt 3 Starter
A frontend starter with nuxt3 and essential plugins.

## prerequis
- node 18

## Plugins installed
- Vite (builder)
- Vuetify 3 (css)
- Material design icons (icons)
- i18n (internationalization)
- Pinia (Store)
- Axios (HTTP client)

## Default components
- App Bar
- Open-close App Menu
- Router
- Dark theme
- Reusable Snack that works with Pinia
